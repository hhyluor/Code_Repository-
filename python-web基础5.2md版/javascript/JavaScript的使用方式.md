# JavaScript的使用方式

**学习目标**

* 能够知道JavaScript的使用方式

---


### 1. 行内式（主要用于事件）

```html
<input type="button" name="" onclick="alert('ok！');">
```
### 2. 内嵌式

```html
<script type="text/javascript">        
    alert('ok！');
</script>
```

### 3. 外链式

```html
<script type="text/javascript" src="js/index.js"></script>
```

### 4. 小结

* JavaScript的使用方式有三种，分别是:
  * 行内式
  * 内嵌式
  * 外链式
