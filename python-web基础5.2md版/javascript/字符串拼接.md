# 字符串拼接

**学习目标**

* 能够实现字符串拼接的操作

---

### 1、字符串拼接

字符串拼接使用: **"+"** 运算符

```js
var iNum1 = 10;
var fNum2 = 11.1;
var sStr = 'abc';

result = iNum1 + fNum2;
alert(result); // 弹出21.1

result = fNum2 + sStr;
alert(result); // 弹出11.1abc
```

**说明**

数字和字符串拼接会自动进行类型转换(隐式类型转换)，把数字类型转成字符串类型进行拼接

### 小结

* **"+"** 运算符能够实现字符串的拼接操作
